# Git Hooks

Этот каталог содержит git hooks для проекта.

## Установка hooks

Для установки hooks в ваш локальный репозиторий выполните:

```bash
# Копирование hooks в .git/hooks
cp .githooks/* .git/hooks/
chmod +x .git/hooks/*
```

Или используйте команду:

```bash
make install-hooks
```

## Доступные hooks

### pre-commit

Проверяет форматирование Go кода перед коммитом:
- Проверяет, что все Go файлы отформатированы через `gofmt`
- Блокирует коммит, если найдены неотформатированные файлы
- Предлагает автоматически отформатировать код

## Автоматическая установка

Если вы хотите, чтобы hooks устанавливались автоматически при клонировании репозитория, добавьте в `.git/config`:

```ini
[core]
    hooksPath = .githooks
```

Или выполните:

```bash
git config core.hooksPath .githooks
```
